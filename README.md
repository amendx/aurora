# Aurora

Gestão de plantões para profissionais de saúde brasileiros.

Calendário inteligente, relatórios financeiros e rede de colegas — integrado nativamente com o PlantaoAPI da WebClient.

## O que é

Aurora é um app React Native / Expo para médicos, enfermeiros e outros profissionais de saúde que trabalham em regime de plantões. Conecta com o PlantaoAPI da WebClient para sincronizar escalas automaticamente, calcula ganhos por plantão e mostra quem está de plantão com você hoje.

## Funcionalidades

- **Calendário de plantões** — mensal, com cores por grupo e turno M/T/N
- **Relatórios financeiros** — valor configurável por hora, adicional noturno, horas extras
- **Colegas de plantão** — veja quem está escalado com você hoje
- **Gestão de grupos** — organize por hospital/clínica, controle visibilidade
- **Integração WebClient** — login com conta do PlantaoAPI, sincronização automática
- **Dark mode** — interface premium que adapta ao ambiente

## Stack

React Native + Expo · Firebase Auth + Firestore · WebClient API (OAuth)
